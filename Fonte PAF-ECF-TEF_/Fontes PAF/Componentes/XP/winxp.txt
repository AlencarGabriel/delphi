Files :
-----
  winxp.zip : component files for Delphi 3,4,5,6 & C++Builder 3,4,5
  

Release 1.0 :
-------------

  Gives your Delphi or C++Builder applications the new visual look of Windows XP
  on a Windows XP machine. 

Feature Revision history :
------------------------

  1/8/2001   : first release


Installation :
--------------

 In Delphi 3,4,5,6 or C++Builder 3,4,5

 Unzip files winxp.zip

 - Install into new package (or existing package)

 - Browse for (new) package file

 - Add winxp.pas
